# 🐔 Balap Ayam — Pixel Racing Game

A fun pixel art chicken racing game built with React + Vite.

Bet on your favorite chicken, watch them race, and win $AYAM!

## 🎮 Play Now

🔗 **Live Demo:** https://chicken-race-seven.vercel.app

## ✨ Features

- 🐔 6 unique pixel art chickens (Common → Legendary)
- 💰 Betting system with multipliers
- 🏁 Animated race with countdown & results
- 🎵 8-bit music & SFX (toggle on/off)
- 🏆 Achievement system
- 📊 Leaderboard with localStorage
- 🎉 Confetti on win
- 📱 Responsive design
- 🔧 PWA installable

## 🛠️ Tech Stack

- React 19
- Vite 7
- TypeScript
- Tailwind CSS
- Web Audio API (music/SFX)
- localStorage (persistence)

## 🚀 Getting Started

```bash
# Install dependencies
bun install

# Run dev server
bun run dev

# Build for production
bun run build
```
